# E-Commerce Intern Debug Tasks

This repository contains 2 separate e-commerce front-end project packs for web developer interns.

Each project contains 5 debugging exercises.
Each exercise includes:
- `index.html`
- `styles.css`
- `script.js`

Project folders:
- `project-01-stylehub`
- `project-02-gadgetgrove`

Instructions for interns:
1. Open one project folder at a time.
2. Run each task in a browser.
3. Inspect the HTML, CSS, and JavaScript.
4. Find and fix the intentional bugs.
5. Document the issues you found and explain your fixes.

Suggested evaluation areas:
- Debugging accuracy
- Code reading and troubleshooting
- HTML/CSS/JavaScript fundamentals
- Attention to detail
- Quality of explanations
